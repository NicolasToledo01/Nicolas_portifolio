Habilidades

Minhas Habilidades

HTML5

HTML5 é a linguagem usada para criar a estrutura básica de sites e aplicativos. Ele define títulos, textos, imagens, botões e organiza todo o conteúdo que aparece na tela, funcionando como o esqueleto de uma página web.

CSS3

CSS3 é a tecnologia responsável por deixar os sites bonitos e organizados. Com ele, é possível escolher cores, fontes, tamanhos, posicionar elementos e criar animações, tornando a experiência visual agradável e moderna.

JavaScript

JavaScript é a linguagem que dá vida e interatividade aos sites. Ele permite criar menus que abrem e fecham, formulários inteligentes, animações e tudo que reage ao clique ou ação do usuário, tornando a página dinâmica.

React.js

React.js é uma ferramenta moderna para construir interfaces de sites e aplicativos. Ele facilita a criação de páginas rápidas, organizadas e que mudam de conteúdo sem precisar recarregar, melhorando a experiência do usuário.

Estudando Agora

TypeScript

TypeScript é um superset do JavaScript que adiciona tipagem estática, tornando o código mais seguro e fácil de manter. Estou estudando para aplicar em projetos mais robustos.

Node.js

Node.js permite rodar JavaScript no backend, criando APIs e aplicações completas. Estou explorando para ampliar meu conhecimento em desenvolvimento fullstack.
